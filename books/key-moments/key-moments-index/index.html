全书目录

前言：为什么写这本书

在信息洪流中建立坐标系

第一章：注意力就是一切

Transformer 架构的诞生，与”大力出奇迹”信仰的建立（2017-2020）

  • Transformer：一篇改变一切的论文
  • BERT vs GPT：两条分岔路
  • GPT-3：涌现的前夜与范式确立

第二章：ChatGPT 时刻

大模型从实验室走向公众，两个月一亿用户（2022-2023）

  • InstructGPT 与 RLHF：被忽视的前奏
  • ChatGPT 发布：史上增长最快的消费级产品
  • GPT-4：能力的跃迁
  • DPO、LIMA 与对齐研究的爆发

第三章：Scaling Laws 的发现与困境

从”大就是好”到”见顶了吗”的认知三部曲（2020-2025）

  • Kaplan Scaling Laws：从”炼丹”到”工程”
  • Chinchilla：参数和数据要平衡
  • 涌现能力的发现与争论
  • 预训练 Scaling 放缓与 test-time compute 的发现

第四章：架构暗战

那些不上头条、却真正决定了谁能赢的技术演进（2018-2025）

  • “现代 LLM 配方”的结晶：Pre-RMSNorm + SwiGLU + RoPE + GQA
  • 位置编码演进：从绝对位置到 RoPE，再到百万级上下文
  • Flash Attention：一个人改变了整个行业的效率
  • 注意力效率之战：MHA → MQA → GQA → MLA
  • MoE：从异端到正统（Switch Transformer → Mixtral → DeepSeek MoE）
  • “小改动，大影响”：RMSNorm、SwiGLU、FP8
  • 数据工程：被低估的胜负手
  • 后训练进化：从 RLHF → DPO → GRPO

第五章：开源觉醒

从 Llama 的泄露到 DeepSeek 的逆袭：开源如何通过架构创新重塑格局（2023-2025）

  • Llama 的泄露与架构标准化
  • Meta 的战略转向：从泄露到拥抱
  • Mixtral：MoE 在开源世界的第一次成功
  • DeepSeek V2/V3：MoE + MLA 的系统性胜利
  • 开源技术生态的爆发

第六章：推理革命

从 o1 到 DeepSeek R1：当模型学会在回答之前「思考」（2024-2025）

  • Test-Time Compute：第二条 Scaling 曲线
  • OpenAI o1：推理模型的横空出世
  • 从 PPO 到 GRPO：关键技术演进
  • DeepSeek R1：纯 RL 涌现推理能力
  • 推理模型成为标配

第七章：从对话到行动

AI Agent 与应用落地的关键突破（2024-2026）

  • 工具使用与 Function Calling
  • Claude Computer Use：AI 操作整个数字世界
  • Coding Agent：最先落地的场景
  • Manus 与 Agent 元年
  • 多模态能力的支撑

第八章：创造力的释放

DeepSeek R1、Vibe Coding 与全民 AI 创造的觉醒（2025）

  • 一个春节改变了一切：R1 的社会冲击
  • 成本革命：免费的力量
  • Vibe Coding：不会编程的人开始编程
  • MCP：AI 的”USB-C 接口”
  • 从”用 AI”到”用 AI 造东西”的认知跃迁
  • OpenAI 被迫开源

第九章：龙虾革命

OpenClaw 与 AI 执行力觉醒：当 AI 真正开始替你干活（2025-2026）

  • 从”能创造”到”能执行”
  • 龙虾的诞生：Clawdbot → Moltbot → OpenClaw
  • 本地优先、CUA、通讯集成、插件生态
  • 春节效应 2.0：从技术圈到全民话题
  • “一人公司”从口号变成现实
  • CUA 的安全隐忧

第十章：未完的战争

大模型的现在与将来：悬而未决的关键问题（2026-）

  • 已经确定的事
  • 悬而未决的关键问题：Agent 可靠性、AGI 距离、就业冲击、平台战争
  • 个人判断与未来校验点
