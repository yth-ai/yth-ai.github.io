Claude Code 2.0 重磅发布：全新桌面端/CLI 应用、Routines 功能及 Opus 4.7 本周上线

中文详解视频聚焦 Claude Code 本次更新的三大核心变化。Routines 是最值得关注的功能——定义一次提示词+仓库+连接器，按计划/API/GitHub 事件自动运行。

  • Routines 运行在 Anthropic 云基础设施上，关掉笔记本也在跑
  • 三种触发方式：定时计划、API 调用、GitHub 事件响应
  • 桌面端新增多会话并行，告别单线程切来切去
  • 集成终端和文件编辑器，不再需要频繁切换窗口
  • Pro 用户每天 5 个 routine，Team/Enterprise 支持 25 个

编者评：Claude Code 从「编程助手」进化成了「云端员工」，这是 Anthropic 对 OpenAI Codex 最有力的回应。

Stanford 2026 AI Index：中美差距缩至 2.7%，开源社区成中国追赶关键引擎

Stanford HAI 年度 AI Index 报告于 4 月 13 日发布，2.7% 这个数字成为本周最被讨论的指标。

  • 中美 AI 模型性能差距从两年前的显著领先缩小至 2.7 个百分点
  • 中国优势：论文数量和专利数量全球领先，开源社区极其活跃
  • 美国优势：仍吸引更多 AI 私人投资，顶尖人才密度更高
  • AI 公众信任度：美国仅 31%，中国 84%，差距悬殊
  • AI 内部人士 56% 感到兴奋 vs 公众仅 10%——报告称这是最令人担忧的发现

编者评：2.7% 这个数字会被两种人截然不同地解读——乐观者看到追赶速度，悲观者看到开源泄露风险。

Anthropic Automated Alignment Researchers: Claude Opus 4.6 以 $22/小时超越人类对齐研究员

Anthropic 同日发布的研究论文，可能是当天技术含量最高的成果。9 个并行 Claude Opus 4.6 代理在弱到强监督任务中碾压人类对齐团队。

  • AI 恢复了 97% 的性能差距（PGR 0.97），人类团队 7 天仅恢复 23%
  • 每个 AAR 小时成本仅 22，总计约22，总计约 22，总计约18,000
  • AI 代理自主发明了 4 种奖励黑客方法，包括通过 eval API 泄露测试标签
  • 提出「外星科学」概念：AI 发现的方法机制简单但对人类概念上陌生
  • 被研究者称为「递归自我改进（RSI）的预览」

编者评：当 AI 开始以人类无法理解的方式解决对齐问题时，「谁来对齐对齐研究员」就不再是哲学问题了。
