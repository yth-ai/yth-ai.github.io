前沿实验室

Anthropic / OpenAI / DeepMind / xAI / Cursor 等

11 ORGS
前沿实验室 US ★

Anthropic

把 RL 环境工业化、把可解释性当一线议题的前沿实验室

Constitutional AI RL Environments Interpretability Long-Horizon Agents

目前所有前沿实验室里最系统化把 RL 环境当成"输入要素"来做的一家——Environment Scaling + Universes 双轨设置已经把 env 提升到和 compute、data 同维度的产业输入。

前沿实验室 US ★

OpenAI

推理时代的领跑者，把 self-play 和合成数据押到 RL 训练核心

Reasoning Models Synthetic RL Self-Play Agent / Computer Use +1

OpenAI 押注的 RL env 路线和 Anthropic 几乎相反——内部建合成数据栈，靠 self-play 和 simulator 而不是外部真实数据供应。这是 AlphaGo 血统的延续。

前沿实验室 US ★

Google DeepMind

Gemini + 物理世界 + 科学 AI 三线并发的老牌实验室

Gemini Robotics AlphaProof / Science World Models

公开 RL env 信号不如 Anthropic/OpenAI 显眼，但 Gemini 后训练背后的 env 栈估计是业内最完整的——只是没有被独立成"Environment Team"品牌。

前沿实验室 US ★

xAI

Musk 系大模型实验室，全内部化数据 + env 不外包的极端路线

Grok Real-time Data Internal Human Data Ops

27 个 Human Data 岗位全部内部消化——这是当前几家前沿实验室里最极端的"垂直整合"。和 Anthropic 完全相反：不要外部供应商，所有数据和 env 都在内部生产。

前沿实验室 US ★

Cursor (Anysphere)

Anysphere / Cursor

AI 优先 IDE 的代表，从工具厂商演化为 neolab 级训练玩家

Coding Agent Internal Models User Data Flywheel

Cursor 已经从单纯工具厂演化成 neolab——靠用户使用数据训练自家专门的 coding 模型。这套"产品即数据飞轮"是 2026 年最值得抄的样本。

前沿实验室 US

Meta AI / FAIR

Llama 系列开源 + 应用产品双线，env 工作分散在 SuperIntelligence Group

Llama Series Open Weights World Models / V-JEPA Robotics

没有公开突出的 env 团队信号，但 Llama 4 后训练背后必然有大量 RLHF + RLVR 工作。Scale AI 部分收购后，env 工作可能通过 Scale 团队承接。

前沿实验室 US

Cognition

Cognition AI

Devin 背后的公司，主打长 horizon coding agent

Devin Long-Horizon SWE Agent Infrastructure

Devin 第一个把"自主 SWE agent"这件事推到产品级。和 Mercor 合作的 APEX-SWE 论文是当前真实 SWE 任务上最严肃的评测之一。

前沿实验室 US

Microsoft Research

Phi 系列 + AutoGen + 大量学术输出，是企业 AI 的隐形巨头

Phi Series AutoGen DevBench Reasoning Research

和 OpenAI 合作 + 自研 Phi/AutoGen 双线。学术输出量在所有公司里是最大的之一，DevBench 等评测工作长期被引用。

前沿实验室 EU

Mistral AI

欧洲前沿实验室，开源 + 商业双线推进

Open Weights Mixtral / Codestral European AI

欧洲前沿实验室代表，开源策略和商业模型并行。RL env 工作公开信息少，但 Codestral 系列说明 coding 方向有持续投入。

前沿实验室 US

Perplexity

Perplexity AI

AI 搜索的代表，Comet 浏览器是 agent 时代的产品级实验

AI Search Comet Browser Sonar

AI 搜索的代名词。Comet 浏览器代理是 2025 年最受关注的 agent 产品之一，但和 Amazon 的法律冲突暴露了"代理购物"路线的真实风险。

前沿实验室 US

Apple Intelligence

Apple Machine Learning Research

端侧 AI + 隐私优先的代表，研究输出比商业声量低调得多

On-Device Models Apple Intelligence Foundation Models Vision Pro

端侧 + 隐私优先的产品形态决定了 Apple 不会发布巨型模型。但研究输出（OpenELM / MM1 / 4M 等）质量稳定，在小模型 + 多模态方向有持续投入。

中国实验室 CN ★

智谱 AI

Zhipu AI

GLM 系列 + AutoGLM agentic 双线推进的中国前沿实验室

GLM Series AutoGLM / Agent Agentic RL

GLM 4.5/4.6 在 agentic 评测上的提升节奏很稳，背后有一整套没有公开讲清楚的 env 栈。AutoGLM 是观察国内 agent 落地的关键样本。

中国实验室 CN ★

DeepSeek

小团队大成果的代表，R1 和 V3 重新定义了开源模型的天花板

DeepSeek R1 / V3 RLVR MoE Cost-Efficient Training

R1 是 2025 年最重要的开源模型 release 之一，把 RLVR 实践拉到全行业关注。技术报告写得相对扎实，是反推 RL 训练方法的好样本。

中国实验室 CN ★

阿里通义实验室

Alibaba Tongyi Lab (Qwen)

Qwen 系列 + AgentScaler 环境路线，中国厂商里 env 思想最系统的一家

Qwen Series AgentScaler Environment Scaling Multimodal

AgentScaler 论文（arXiv:2509.13311）是国内对 RL env 最有理论自觉的产出——程序化构建海量异构环境让 4B 模型对齐万亿级。这条路对算力受限玩家有战略价值。

中国实验室 CN

Moonshot AI / Kimi

Moonshot AI

长上下文起家，Kimi K2 之后向 agentic 方向发力

Kimi Long Context Agentic

长上下文起家，K2 之后明显加大 agentic 投入。开源策略偏保守，但产品端 Kimi 用户基数和反馈链是别家少见的资产。

中国实验室 CN

阶跃星辰

StepFun

Step Series Multimodal Voice / Vision

多模态投入早且密，特别是语音和视频方向。RL 类工作披露较少，但开源版 Step 系列质量稳定。

5 ORGS
RL 基础设施 US ★

Prime Intellect

开源 full-stack RL 训练栈：从 PRIME-RL 到 Environments Hub 一应俱全

PRIME-RL Verifiers Library Environments Hub Distributed RL +1

目前社区版 OpenAI training infrastructure 的最强候选。同时拥有训练框架（PRIME-RL）、env 库（Verifiers）、沙箱、Hub、托管平台五件套，是开源 RL 栈最完整的一家。

RL 基础设施 US ★

General Reasoning

试图把 RL 环境协议化的玩家，OpenReward + ORS 是最重要的产品

OpenReward Open Reward Standard (ORS) Reward Engineering

Ross Taylor（前 Meta AI / Llama 2-3 研发负责人 / Galactica 第一作者）创办，目标是把 RL env 变成像 HTTP 一样的标准化基础设施。如果 ORS 成事实标准，整个产业的价值链会被重新切。

RL 基础设施 US

Mechanize

把"自动化所有人类工作"作为目标的小团队，env / RL 经济学的关键发声者

Automated Tasks RL Economics Long-Horizon Agents

Mechanize 的博文密度不高，但篇篇都触及 RL env 经济学的关键问题——比如"廉价 RL 任务会浪费算力"那篇直接给出了 $2,400/task 的训练成本估算，是产业讨论的高频引用。

RL 基础设施 US

Together AI

推理 + 训练云的代表，Together Endpoint 是最广泛使用的开源模型推理服务

Inference Cloud Fine-tuning RedPajama

开源模型推理云的事实标准。RedPajama 数据集和 OpenChatKit 是早期对开源生态的重要贡献。当前更偏推理服务，但训练相关工作没有停。

RL 基础设施 US

Databricks (Mosaic)

Databricks Mosaic AI

收购 MosaicML 后成为企业 AI 训练 + 推理的主要供给方

Enterprise AI Training Infrastructure DBRX

Mosaic 收购后，企业内部训练 / 微调的主要服务商。DBRX 模型 release 不算频繁，但企业 AI 商业化层面的影响力很大。

数据供应商

Scale / Surge / Mercor / Handshake / Turing

5 ORGS
数据供应商 US

Scale AI

数据标注起家，正在向 eval-driven RL 和 rubric reward 转型

Data Labeling Eval / RL Rubrics as Rewards SEAL Leaderboards

Meta 部分收购之后处在转型期。研究端 ICLR 2026 接收 6 篇论文（含 RaR），主张 "verification asymmetry 是下一个关键瓶颈"——这个判断本身比公司动作更重要。

数据供应商 US

Surge AI

标注起家，CoreCraft 把模拟职场变成可训练 env 的代表

Annotation Hierarchy of Agentic Capabilities EnterpriseBench / CoreCraft

Surge 的研究产出在数据公司里水位很高。Hierarchy of Agentic Capabilities + CoreCraft 是少见的"把真实职场任务变成 RL env"的样本，训练后 GLM 4.6 在 OOD 基准上的迁移数据是行业级证据。

数据供应商 US

Mercor

专家招募 + RL 任务交付的代表，APEX-SWE 评测背后的力量

Expert Network SWE Tasks APEX-SWE

从招专家做标注，扩展到帮前沿实验室构建 SWE / RL env。APEX-SWE（与 Cognition 合作）的 Pass@1 40.5% 是当前真实 SWE 任务上模型水位的关键基准。

数据供应商 US

Handshake

Handshake AI

校园招聘转型 AI 数据，主打学生专家网络做 RL 训练任务

Student Expert Network RL Tasks

Handshake 把原本服务校招的网络转型到 AI 数据/RL env 任务。供给侧最大的特点是触达大量在校学生专家，能在 STEM 任务上快速规模化。

数据供应商 US

Turing

软件工程师匹配平台转型，主攻 SWE / coding 类训练数据

SWE Data Coding Tasks Long-Horizon Code RL

原本做远程程序员匹配，2024 年起转型成前沿实验室的 coding RL 数据供应商。SWE 类任务深度是它的差异化。

3 ORGS
开源生态 US ★

Hugging Face

开源 AI 生态的中枢——Hub、Transformers、Datasets 一应俱全

Model Hub Datasets Transformers Open Reasoning Tools

开源生态的事实标准。Hub 上 model / dataset 的发布节奏比官方博客快得多，做行业跟踪几乎绕不开。

开源生态 US

AI2 (Allen Institute for AI)

Allen Institute for AI

完全开源训练栈的代表——OLMo / Tülu / Dolma 系列把"训练复现"做到极致

OLMo Tülu Dolma Reproducible Training

AI2 是少数把"训练栈完全开源"做成业务的研究机构。OLMo / Tülu / Dolma 三件套覆盖模型 + 后训练 + 数据，是研究复现的最佳起点。

开源生态 US

EleutherAI

开源大模型生态的早期推动者，lm-evaluation-harness 是事实标准评测框架

GPT-NeoX Pythia lm-evaluation-harness Interpretability

非营利研究社区，长期产出对学术界很重要的基础工具——lm-eval-harness 是几乎所有公开榜单的底座。Pythia 系列是研究"训练动态"的标准模型套件。

5 ORGS
研究机构 US ★

Epoch AI

数据驱动的 AI 产业研究机构，Gradient Updates 是行业最稳定的解读源之一

AI Trends Compute / Data Analysis RL Environments Industry Research

独立、数据驱动、不卖产品的研究机构。Gradient Updates 是产业判断的高质量信号源——FAQ on RL Environments 和 AI Lab Job Postings 两篇是 RL env 行业讨论的标准引用。

研究机构 US

SemiAnalysis

AI Compute Hardware Datacenter Economics RL Infra

半导体 + AI 算力交叉地带最有触觉的 newsletter。RL env 经济学、单 env 报价、产业供应链等等几乎只能在这里看到一手细节。

研究机构 US

LMSYS

LMSYS (Large Model Systems Org)

Chatbot Arena 的运营方，模型偏好评测的事实标准

Chatbot Arena SGLang Vicuna

Chatbot Arena 是当下最被引用的人类偏好评测，几乎所有 frontier model release 都会拿 Arena 排名做背书。SGLang 是高性能推理引擎的代表。

研究机构 US

NVIDIA Research

模型 + 训练框架 + 数据集三线并发的"算力厂的研究部门"

Nemotron Megatron Synthetic Data Robotics Sim

Nemotron 系列模型 + 大量公开训练数据 + Isaac Sim 三件套。NVIDIA 在 RL env 上的影响主要通过仿真平台和合成数据集传导。

研究机构 US

METR

METR (Model Evaluation & Threat Research)

专注 AI 危险能力评估的独立研究机构，Long-Task Horizon 评测代表作

Dangerous Capabilities Eval Long-Task Horizon Autonomy Research

METR 把"自主任务时长"作为衡量 AI 进步的关键指标——这个视角非常有解释力。Long-Task Horizon 系列论文是当下评估 agent 能力的少见方法。
