## 阿里「欢乐马」匿名空降屠榜

**Two Minute Papers 拆解 HappyHorse-1.0 匿名屠榜事件**

4 月 8 日，一个没有发布会、没有技术博客、甚至没有公司背书的匿名视频模型 HappyHorse-1.0 突然空降 Artificial Analysis Video Arena 榜首，ELO 1365 大幅碾压第二名 Seedance 2.0 和 Kling 3.0。Two Minute Papers 从多个盲测案例切入，展示了这匹「快乐马」在画面真实感、物理一致性和细节保留方面的惊人表现。

- 文本转视频和图像转视频两项赛道均登顶，15B 参数规模在效率上令人惊讶
- 48 小时内从「全网竞猜」到阿里 ATH 事业群前千问负责人林俊旸团队正式认领
- 开源策略——预计一周后公开模型权重，延续阿里「调用量全球领跑」的开源路线
- 背后或基于 daVinci-MagiHuman 优化，统一框架处理文本/图像/视频三种输入

**编者评论**：匿名先屠榜再认领，这是阿里继通义千问之后又一次精心策划的开源攻势——用实力说话，先让数据替品牌发声。

## OpenAI Spud 六层超级应用生态

**Fireship 全面拆解 Spud 不止于模型本身**

距 GPT-6「土豆」传闻的 4 月 14 日发布日仅剩 3 天，Polymarket 给出 78% 的概率。Fireship 这期不聚焦模型参数，而是拆解 OpenAI 围绕 Spud 构建的完整超级应用生态——从桌面到汽车再到耳机的全场景环境计算。

- 六层架构：Spud（核心智能）→ Codex CLI（多 Agent 开发）→ File Library（跨会话记忆）→ Apple CarPlay（1 亿辆车）→ TBPN 收购（内容分发）→ Sweet Pea 耳机（9 月发布，首年目标 4-5 千万台）
- Codex 周活突破 300 万（上月 200 万），VB 在 ThursdAI 透露自动化早晨简报工作流
- Greg Brockman 播客原话：「两年研究，大模型的感觉，不是增量改进」——强烈暗示这是 GPT-6 而非 GPT-5.5
- 8520 亿美元估值压力下，Spud 必须是突破性产品，不能重蹈 GPT-4.5「令人失望」的覆辙

**编者评论**：OpenAI 的野心已经从「做最好的模型」升级为「做最好的生态」。Sweet Pea 耳机如果成功，这将是 AI-first 硬件的真正起点。

## 字节 Seeduplex 全双工语音革命

**TheAIGRID 解析「边听边说」如何终结回合制**

4 月 9 日，字节跳动 Seed 团队发布原生全双工语音大模型 Seeduplex，已在豆包 App 全量上线。TheAIGRID 从技术架构到用户体验全面解析这一突破。

- 核心创新：从半双工「你说我听、我说你听」升级为「边听边说」——AI 在生成回复的同时持续监听用户输入
- 嘈杂环境精准抗干扰：车内、咖啡厅等复杂声学场景下依然精准锁定用户语音
- 情绪感知与自然打断：能识别用户语气变化并自然响应打断，对话体验接近人类
- 豆包日耗 1.6 万亿 Token 的底座上部署，国产语音交互进入新阶段

**编者评论**：全双工是语音 AI 从「工具」到「伙伴」的分水岭。当 AI 能像人一样边听边想边说，回合制对话将成为历史。

## Claude Code 一周年回顾

**编者评论**：一年前 Claude Code 还是个实验品，今天它已经改变了数百万开发者的日常工作流。Agent 编程的速度比所有人预想的都快。

## AI 供应链安全警钟

**Swyx 在 X 发起的 fork-not-install 讨论**

AI Engineer 创始人 Swyx 在伦敦大会后发推总结 Agent 生态的供应链安全隐患，引发广泛讨论。

- light LLM 和 Axios 接连爆出供应链注入漏洞，Agent 工具链的依赖深度远超传统 web 应用
- OWASP 2026 年版 Agentic Applications Top 10 将供应链攻击列为核心威胁
- 建议：fork 核心依赖包而非直接 npm install，尤其是 Agent 框架的 MCP 桥接层
- The Register 4 月 10 日长文梳理：AI 安全已从「实验阶段」进入「代理时代」

**编者评论**：当 Agent 能自主调用工具链时，一个被污染的依赖包不再只是代码漏洞——它是 AI 的「被操纵入口」。

## 编者按

本期前沿声浪的主线是一个词：**生态**。

OpenAI 不再满足于做最好的模型，它要用 Spud 串联起从桌面到汽车到耳机的全场景环境计算；阿里用 HappyHorse 的匿名屠榜证明开源生态的杀伤力——先让产品说话，再让品牌收割；字节的 Seeduplex 把语音 AI 从回合制推进到全双工，豆包的日耗 Token 量已经是全球之最。

而在这些光鲜的产品背后，ThursdAI 伦敦圆桌揭示了一个更深层的真相：**算力决定一切**。Peter Gostev 的 Compute Wars 图表说明了为什么 Anthropic「不敢」发布 Mythos，为什么 Meta 要从零建 Muse Spark 的基础设施。模型之争的本质是算力之争，算力之争的本质是资本之争。

同时，Swyx 的供应链安全警钟提醒我们：当 Agent 变得越来越强大、工具链越来越深，安全问题也从「模型对齐」扩展到了「生态安全」。一个被污染的 npm 包，在 Agent 时代可能比一个未对齐的模型更危险。
